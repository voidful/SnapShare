Linking Your Phone To Computer Browser With Socket.io. 

## Feature
- share link without manually copy long url.
- no set up and download is needed for user.
- Front-end React.
- Back-end  Flask.
- Connect with Socket.io.

## Get Start
1. prepare Docker in your environment
2. Start Containers
```
docker-compose up
```
